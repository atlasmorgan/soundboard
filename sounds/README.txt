HOW TO ADD SOUNDS TO YOUR SOUNDBOARD
=====================================

1. Find free MP3 sound files online. Great free sites:
   - https://freesound.org  (search for any sound, free download)
   - https://www.myinstants.com  (tons of meme sounds!)
   - https://zapsplat.com  (free with account)

2. Download the MP3 file and rename it to match these filenames:
   - vine-boom.mp3
   - bruh.mp3
   - oof.mp3
   - airhorn.mp3
   - nyan.mp3
   - laugh.mp3
   - sad-trombone.mp3
   - wilhelm.mp3
   - roundabout.mp3
   - fart.mp3
   - prank.mp3
   - turtle.mp3

3. Put the renamed MP3 files in THIS folder (the sounds/ folder).

4. Open index.html in your browser and click the buttons!

TIPS:
- If a button doesn't make a sound, the MP3 file might be missing.
- The file names must match EXACTLY (lowercase, with dashes).
- Chrome and Firefox work best for playing sounds.

Have fun! 🎉
